# AI-Based Corporate Attendance and Monitoring System

## Project Overview

AI-Based Corporate Attendance and Monitoring System is a web-based application that automates employee attendance management using Artificial Intelligence and Computer Vision.

The system allows users to register, login, upload face images, detect faces using OpenCV, and mark attendance automatically. Attendance records are stored in SQLite database and CSV format.

---
